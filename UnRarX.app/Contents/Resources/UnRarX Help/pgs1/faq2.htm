UnRarX cannot write to File Directory?

This simple step by step process will help you get started extracting rar archives when permissions prevent you from writing to the File Directory that contains your rar archive.

Note: In some cases, depending on the permission's settings on your computer UnRarX is unable to extract files from your rar archive. My advice in this situation is to try extracting the files to your "Desktop" instead of the "File Directory". This situation is likely to happen if you are trying to extract from a CD or a read-only drive.

1. Launch UnRarX.

2. Using "Extract to" pulldown select "Desktop"

3. Drag rar archive into the UnRarX window.

4. Extraction begins automatically.

There are many different rar sequences. For rar sequences that include part numbers (i.e. part001.rar, part002.rar) using the first rar archive in the sequence gives the best results.
